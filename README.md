## Setup

### Strapi backend

- [Check requirements.](https://docs.strapi.io/dev-docs/quick-start)

  - Node version 18 or 20
  - npm or yarm

- Switch Docker Desktop to Linux containers

- Open PowerShell 7 as an Admin

- `cd strapi-backend`

- Run `docker-compose up -d` to run containers.

- Strapi Admin runs in http://localhost:1337/admin

### Frontend

- `cd frontend`

- `npm i`

- If u need to install Angular CLI `npm install -g @angular/cli`

- `npm run start`
